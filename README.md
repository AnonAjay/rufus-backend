Rufus — Backend
This is the official repository for rufus-backend, part of the Rufus companion robot dog family — the FastAPI backend, agent orchestration (LangGraph + LiteLLM), and MCP tool servers that power Rufus's decision-making, device communication, and third-party integrations.
If you find this project useful or interesting, please consider giving it a ⭐ — it helps others discover the project and keeps us motivated to keep building.
Contributing: We welcome collaboration! Before submitting a pull request, please reach out to the maintainers first to discuss the issue and proposed solution — this helps us align on approach and avoid duplicate or conflicting work.
